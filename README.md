# 🏠 КовёрМастер — AI Telegram Бот

Система управления бизнесом по мойке ковров с забором и доставкой.

## Что умеет бот

### Для владельца (администратора):
- 📋 Просмотр всех заказов с фильтрами
- ➕ Создание заявок пошагово
- 🔄 Смена статусов заказов одной кнопкой
- 🚗 Автоназначение водителей
- 📊 Статистика: выручка, заказы, клиенты
- 💬 AI-консультант по любым вопросам бизнеса
- 📩 Автоуведомления клиентам при смене статуса
- 📊 Дневной AI-отчёт в 20:00

### Для клиентов:
- /order — оставить заявку
- /status — статус своих заказов
- /price — расчёт стоимости
- Поддержка AI 24/7

## Быстрый старт

### 1. Получите необходимые ключи

**Telegram Bot Token:**
1. Найдите @BotFather в Telegram
2. Отправьте /newbot
3. Следуйте инструкциям, получите токен

**Ваш Telegram ID:**
1. Напишите @userinfobot
2. Скопируйте цифровой ID

**Anthropic API Key:**
1. Зайдите на https://console.anthropic.com
2. Создайте API key

### 2. Установка

```bash
# Клонируйте или скачайте папку бота
cd carpet_bot

# Установите зависимости
pip install -r requirements.txt

# Создайте .env файл
cp .env.example .env

# Откройте .env и заполните ваши данные
nano .env
```

### 3. Запуск

```bash
# Запуск бота
python bot.py
```

### 4. Деплой на Railway (бесплатно)

1. Зайдите на https://railway.app
2. New Project → Deploy from GitHub (или загрузите папку)
3. Добавьте переменные окружения из .env
4. Railway автоматически запустит бот

## Структура заказа (статусы)

```
🆕 новый
  ↓
🚗 забрали (водитель забрал ковёр)
  ↓
🧹 в чистке (ковёр на производстве)
  ↓
✅ готов к доставке
  ↓
📦 доставлен (закрытый заказ)
```

При каждой смене статуса → клиент получает уведомление в Telegram.

## Масштабирование (следующий шаг)

Когда бизнес вырастет:
- Замените SQLite на PostgreSQL (одна строка в database.py)
- Добавьте веб-панель (Flask/FastAPI)
- Подключите онлайн-оплату (ЮKassa, Stripe)
- Добавьте фото ковров (до/после)
- Интеграция с картами для оптимизации маршрутов
